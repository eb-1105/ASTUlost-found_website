🚀ALFS | ASTU Lost & Found System is a high-performance, mobile-first web ecosystem designed to bridge the gap between lost items and their rightful owners. Built with a focus on User Experience (UX), Micro-interactions, and Campus Safety Engineering.
Engineering HighlightsApple
Inspired UI/UX: Features a hardware-accelerated iPhone mockup with a "Dynamic Island" interface.
Logic-Based AI Assistant: An integrated chatbot with an intent-recognition decision tree to guide students through the recovery process.
Performance Optimized: Zero-dependency Vanilla JS ensures a near-perfect Google Lighthouse score (FCP < 1.0s).
Fluid Animations: GPU-accelerated shimmer effects, typing animations, and scroll-reveal transitions.
Responsive Architecture: Fully adaptive layout from 4K monitors to mobile devices.
Dual-Theme Engine: Native Dark/Light mode support with smooth 500ms transitions.

🤖Integrated AI Assistant

The system features an ALFS Core AI that utilizes an optimized decision tree to handle student queries:

Intent: ID Card → Action: Redirects to Security Block 501 protocols.

Intent: High-Value → Action: Triggers serial number verification prompts.

Intent: Reporting → Action: Live-links to the incident reporting engine.
